# Django Modular Inventory System

A modular Django REST API for Inventory Management.

## Modules Included:
- **Vendor:** Management of suppliers.
- **Product:** Inventory and stock tracking.
- **Course & Certification:** Educational module integration.

## Setup:
1. Extract the `intern_assignment (2).zip` file.
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
5. Documentation: View Swagger UI at `/swagger/`
